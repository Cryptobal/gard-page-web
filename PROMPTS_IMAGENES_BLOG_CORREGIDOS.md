## Instrucciones de Uso

### Paso 1: Acceder a Leonardo.ai
1. Ve a: https://leonardo.ai
2. Regístrate gratis con Google o email
3. Recibes 150 créditos diarios gratis

### Paso 2: Configurar Generación
```
Modelo: Leonardo Phoenix (mejor para realismo)
Dimensiones: 1920 x 1080 (horizontal)
Número de imágenes: 4 variaciones
Contrast: Medium
Quality: High
```

### Paso 3: Generar
1. Copia el prompt que quieras
2. Pégalo en el campo de texto
3. Click en "Generate"
4. Espera 30-60 segundos
5. Descarga la mejor de las 4 variaciones

### Paso 4: Optimizar
1. Ve a: https://squoosh.app
2. Sube la imagen
3. Selecciona formato: WebP
4. Ajusta calidad: 85%
5. Descarga (debe quedar <500KB)

### Paso 5: Subir a Cloudflare
1. Dashboard Cloudflare → Images
2. Upload la imagen optimizada
3. Copia el Image ID generado
4. Reemplaza en el archivo .md:

```markdown
# Antes:
imageId: "seguridad-eventos-masivos-chile-2025"

# Después:
imageId: "abc123def-456-789-xyz"
```

## Consejos para Mejores Resultados

### Si el uniforme NO sale correcto:
Agrega al final del prompt:
```
, especificar: uniforme de guardia chileno con pantalón y camisa negros, chaleco reflectante de alta visibilidad color rojo brillante con franjas plateadas reflectantes, letras blancas "SEGURIDAD PRIVADA" en la espalda del chaleco, estilo uniforme oficial Chile
```

### Si necesitas más variaciones:
- Agrega: "ángulo frontal" o "ángulo lateral" o "vista de espalda"
- Cambia: "fotografía documental" por "fotografía editorial" o "fotografía comercial"
- Modifica iluminación: "luz natural" o "luz de estudio" o "luz dramática"

### Para diseños gráficos más minimalistas:
Agrega al final:
```
, estilo minimalista profesional, máximo 4 colores, diseño limpio sin saturación, composición balanceada
```

## ¿Problemas?

### La imagen tiene elementos incorrectos:
1. Regenera con el mismo prompt (da resultados diferentes)
2. O usa el prompt alternativo (fotografía ↔ diseño gráfico)

### El uniforme no se ve bien:
1. Usa el prompt de "Fotografía Realista"
2. Agrega el texto adicional del uniforme que puse arriba
3. En Leonardo.ai selecciona "PhotoReal" como estilo

### La calidad es baja:
1. En Leonardo.ai: aumenta "Quality" a máximo
2. Genera en dimensiones más grandes: 2048x1152
3. Luego redimensiona a 1920x1080 en Squoosh
