## Configuraciones Recomendadas por Herramienta

### Midjourney (Discord)
```
/imagine prompt: [PEGAR PROMPT AQUÍ]

Configuraciones adicionales:
--ar 16:9 (ratio horizontal)
--v 6 (versión 6, mejor calidad)
--style raw (más realista)
--q 2 (calidad máxima)
```

### DALL-E 3 (ChatGPT Plus)
```
Simplemente pega el prompt completo.
DALL-E 3 es muy bueno entendiendo español, así que puedes usar los prompts en español directamente.

Configuración:
- Selecciona "Wide" (1792x1024)
- Calidad: "Standard" es suficiente
```

### Leonardo.ai (Web)
```
1. Selecciona modelo: Leonardo Phoenix o Leonardo Diffusion XL
2. Dimensiones: Custom → 1920x1080
3. Pega el prompt en inglés
4. Configuración:
   - Contrast: Medium
   - Generation Mode: Quality
   - Num Images: 4
```

### Stable Diffusion (Local/Web)
```
Modelo recomendado: SDXL 1.0 o Realistic Vision V5.1

Configuraciones:
- Size: 1920x1080
- Sampling Steps: 30-40
- CFG Scale: 7-9
- Sampler: DPM++ 2M Karras

Negative prompt (agregar a todos):
"blurry, low quality, distorted, ugly, amateur, watermark, text, logo, signature, bad anatomy, deformed"
```

## Post-Procesamiento

### Después de generar las imágenes:

1. **Redimensionar si es necesario:**
   - Tamaño final: 1920x1080px exacto
   - Herramienta: Photoshop, GIMP, o web (squoosh.app)

2. **Convertir a WebP:**
   - Comando: `cwebp input.jpg -q 85 -o output.webp`
   - O usar: squoosh.app (web)

3. **Optimizar peso:**
   - Objetivo: <500KB por imagen
   - Calidad: 80-85%

4. **Subir a Cloudflare Images:**
   - Dashboard Cloudflare → Images
   - Upload las 6 imágenes
   - Copiar los Image IDs generados

5. **Actualizar archivos markdown:**
   - Reemplazar los `imageId` placeholders
   - Ver archivo: ESPECIFICACIONES_IMAGENES_BLOG.md

## Consejos para Mejores Resultados

### Si la imagen NO sale bien:

1. **Regenerar con ajustes:**
   - Agrega: "photorealistic, professional photography"
   - Quita elementos que no salieron bien
   - Simplifica el prompt

2. **Usar prompts alternativos:**
   - Cada artículo tiene 2 opciones: fotografía o diseño gráfico
   - Prueba ambas y elige la mejor

3. **Combinar con logos/branding:**
   - Genera la imagen base
   - Agrega logo Gard en post-procesamiento (Photoshop/Canva)
   - Esquina superior o inferior, discreto

4. **Iteración:**
   - Genera 4-8 variaciones de cada prompt
   - Selecciona la mejor
   - Refina si es necesario
